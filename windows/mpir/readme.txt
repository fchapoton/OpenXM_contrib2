The MPIR is an open source multiprecision integer (bignum) library
forked from the GMP (GNU Multi Precision).  This directory contains
pre-built binaries of the MPIR 2.6.0 by Visual Studio 2010.  The
source is distributed in http://www.mpir.org/.
