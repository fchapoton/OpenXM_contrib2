This directory contains pre-built binaries of MPIR 2.7.0, GNU MPFR 3.1.3 
and GNU MPC 1.0.3 compiled by Visual Studio 2013.

For getting the sources, please visit the following sites:
    http://www.mpir.org/
    http://www.mpfr.org/
    http://www.multiprecision.org/
